## üìö What is TTM Squeeze?

The TTM Squeeze is a powerful volatility and momentum indicator that identifies consolidation periods followed by explosive breakouts.

### Strategy Logic

**üü¢ Squeeze ON (Consolidation)**
- Bollinger Bands compressed inside Keltner Channels
- Market building energy
- **Action:** Wait for breakout

**üî¥ Squeeze FIRE (Breakout)**
- Bollinger Bands expand outside Keltner Channels  
- Volatility explosion
- **Action:** 
  - üöÄ **BUY** if Momentum > 0
  - üîª **SELL** if Momentum < 0

## ‚öôÔ∏è Configuration

Edit `config.py` to customize:

```python
# Monitored pairs
MONITORED_PAIRS = ['EURUSD', 'GBPUSD', 'USDJPY', ...]

# Timeframe
DEFAULT_INTERVAL = '1h'  # '15m', '30m', '1h', '4h', '1d'

# TTM Squeeze parameters
BB_PERIOD = 20
BB_STD_DEV = 2.0
KC_PERIOD = 20
KC_ATR_MULTIPLIER = 1.5

# Scheduler (for 24/7 monitoring)
CHECK_INTERVAL = 900  # 15 minutes
```
